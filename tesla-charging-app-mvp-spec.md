# Tesla 한국형 충전 최적화 앱 MVP 기능명세서

작성일: 2026-03-10

## 1. 제품 한 줄 정의
한국 테슬라 운전자를 위해, **현재 배터리 상태와 한국 DC콤보 충전 인프라를 반영해 장거리 충전 경로를 최적화**해주는 앱.

---

## 2. 문제 정의
한국 고속도로 장거리 주행에서 테슬라 운전자는 다음 문제를 겪는다.

- 슈퍼차저가 최적 위치에 항상 있는 것이 아님
- 휴게소 DC콤보를 활용해야 하는 경우가 많음
- 현재 SoC 기준으로 어디를 경유해야 안전한지 판단이 어려움
- 도착 잔량 예측이 불안정함
- 프리히팅이 슈퍼차저 중심 UX라 비슈퍼차저 활용 경험이 매끄럽지 않음

---

## 3. MVP 목표
MVP는 다음 질문에 답해야 한다.

1. 지금 배터리로 어디서 충전해야 하지?
2. 어느 휴게소 DC콤보를 경유하는 게 가장 효율적이지?
3. 거기 도착하면 몇 % 남지?
4. 몇 분 충전하고 다시 출발해야 하지?
5. 프리히팅은 언제 유도하는 게 좋지?

---

## 4. 타깃 사용자

### 1차 타깃
- 한국 테슬라 오너
- 장거리 주행 빈도 높은 사용자
- 고속도로 휴게소 DC콤보를 자주 쓰는 운전자

### 2차 타깃
- 여행/출장이 잦은 EV 사용자
- 충전 스트레스가 큰 초중급 테슬라 운전자

---

## 5. 핵심 MVP 기능

### A. 차량 연결
- Tesla 계정 연동
- 현재 차량 선택
- 현재 SoC 읽기
- 차량 wake 상태 확인

### B. 경로 입력
- 출발지 입력
- 목적지 입력
- 현재 위치 사용 옵션
- 도착 목표 잔량 옵션(예: 10%, 15%, 20%)

### C. 충전소 추천
- 한국 DC콤보 충전소 후보 계산
- 경로상 경유 후보 1~3개 추천
- 우회 비용 최소화
- 충전소별 리스크 점수 표시(향후)

### D. 에너지/도착 잔량 계산
- 현재 SoC 반영
- 차종 반영
- 외기온도 반영(초기 간단 모델)
- 예상 도착 SoC 계산
- 구간별 충전 필요 여부 표시

### E. 충전 전략 제안
- 추천 충전소 도착 예상 SoC
- 몇 분 충전해야 하는지
- 몇 %까지 충전 후 출발할지
- 다음 구간 도착 예상 SoC

### F. 프리히팅 보조 UX
- 도착 전 프리히팅 권장 타이밍 안내
- 차량 내 네비 목적지 지정 권장 문구
- 추후 API 검증 시 자동화 확장 여지 확보

---

## 6. MVP에서 제외할 것
초기 버전에서는 아래를 제외한다.

- 비슈퍼차저 DC콤보 자동 프리히팅 보장
- 커뮤니티 후기/평점 시스템
- 가족 차량 공유
- 충전 시작/중단 자동 제어
- 다중 경유지 자동 최적화 고급 모드
- 완전한 실시간 혼잡도 예측

---

## 7. 사용자 플로우

### 플로우 1: 장거리 주행 시작 전
1. 앱 실행
2. Tesla 계정 연동 / 차량 선택
3. 현재 SoC 자동 반영
4. 출발지/목적지 입력
5. 추천 경유 충전소 확인
6. 경로 선택

### 플로우 2: 추천 결과 확인
1. 추천 경유지 1~3개 표시
2. 각 경유지의 장단점 비교
3. 충전 시간/도착 SoC 비교
4. 프리히팅 권장 시점 안내

### 플로우 3: 실제 주행 중
1. 주행 진행
2. 남은 잔량/조건 변화 시 추천 재계산
3. 필요 시 대안 충전소 제안

### Screen 3. 추천 결과
- 추천 경유지 카드
- 도착 SoC
- 충전 시간
- 최종 도착 SoC
- 프리히팅 권장 타이밍

### Screen 4. 경로 상세
- 구간별 거리
- 경유 충전소 위치
- 충전 전/후 SoC
- 대체 경로 비교

### Screen 5. 설정
- 차종 설정
- 전비 보정값
- 알림 설정
- 구독/프리미엄

### 제품 지표

- 재방문율
- 유료 전환율

### 품질 지표

---

## 12. 프리미엄 방향

### 무료
- 기본 경로 추천
- 후보 1~2개 제시
- 기본 도착 잔량 표시

### 유료
- 고급 경로 최적화
- 온도/고도 반영 강화
- 대안 경로 비교
- 개인 전비 프로파일
- 프리히팅 타이밍 고급 안내
- 주행 중 추천 재계산

---

## 13. 리스크
- Tesla 공식 API 범위 제한
- 목적지 전송/프리히팅 자동화 불확실성
- 한국 충전소 데이터 품질 이슈
- 실제 도착 SoC 예측 정확도 확보 난이도

---

## 14. MVP 개발 우선순위

### P0
- 차량 연동
- SoC 읽기
- 출발/목적지 입력
- 충전소 추천
- 도착 SoC 계산
- 충전 전략 추천

### P1
- 외기온도 반영
- 차종별 전비 보정
- 프리히팅 권장 타이밍
- 대안 경로 비교

### P2
- 실시간 재계산
- 실제 주행 로그 기반 개인화
- 충전소 신뢰도/혼잡도 반영

---

## 15. 한 줄 정리
MVP의 본질은 **프리히팅 자동화 앱**이 아니라,
**한국형 테슬라 장거리 충전 최적화 앱**이다.
프리히팅은 차별화 포인트지만, 초기엔 보조 UX로 다루는 것이 안전하다.

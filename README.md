# Supercombo

Make highway DC combo chargers work more like Superchargers for Tesla drivers in Korea.

## What's in this repo
- `ChargeRouteKR/` — SwiftUI iPhone app skeleton
- `tesla-charge-data/` — charger data normalization pipeline
- `tesla-charge-mockup/` — local web mockup
- product/research docs for feasibility, MVP, preconditioning PoC, and data pipeline

## Core idea
Supercombo aims to help Tesla drivers in Korea use highway DC combo chargers more like Superchargers by combining:
- route-aware charging decisions
- energy / arrival SoC estimation
- preconditioning compatibility research
- Korea-specific highway charger data
